RATING: 8
This is a re-skin of the latest version of Hare & Tortoise with the event cards, and the spaces set to 80. A charming, well thought-out, math-y, staggering race. Wonderful production and artwork. Warning: the money cards are small.

2-player variant is very good. I prefer this to 2-player Ave Caesar and 2-player Cartegena.

A downside I can say is that the board of this version 1st edition is more function than form in that I don't feel that I'm travelling around the world, despite that some of the spaces announce city names. Another downside is that the money could've been done better with tokens and a player screen instead.

RATING: 6
A neat racing game that is a nice remake of the old classic.  I don't recall the old game well, but I think this one made some frustration-reducing changes to reduce luck and focus more on money management.  The pressure between too much money, too little money, and second guessing rank order keeps the game interesting.

Players do need to be brisk, despite the temptation to over-calculate (which is not as useful as it appears, but tempting none-the-less).  If downtime is low, this is fun.  If other players overthink it, the game can quickly become tedious.
