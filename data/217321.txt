Deja-vu, der Name ist Programm.
Es werden einzeln Karten aufgedeckt und wenn man der Meinung ist ein Symbol bereits gesehen zu haben, schnappt man es sich aus der Auslage. Falls es wirklich das zweite Symbol war ist das okay, falls das Symbol im verlaufe doch noch aufgedeckt wird ist dieser Spieler raus.
Sobald der Stapel (abgesehen von 3 Karten) durchgespielt ist endet die Runde und jeder Spieler erhält so viele Punkte, wie Marker vor einem liegen, bzw. 0 falls er aus dem Spiel ist.
Das ganze wird 3 mal gespielt und dann abgerechnet.
Von Runde zu Runde wird es schwerer zu erkennen ob ein Ding bereits ausgelegt wurde oder ob das in der vorherigen Runde passiert ist. Ich bin 3 mal aus dem Spiel geflogen, es hat dennoch viel Spaß gemacht.

RATING: 6
This is an interesting concentration/memory variant in that, instead of looking for a matching pair, you grab the matching token from a token pool when you see the picture twice when revealed from a deck of cards. 

Adding more to this variant is that 3 cards are removed from the beginning, so you can't just grab all the tokens as you are penalised for taking a token before it appears the second time or it appears on the 3 cards set aside.

Much like Jungle Speed, Deja Vu tries to slip you up by making alot of the items appear similiar in shape and colours (lightbulb and pear for example), so I really like that the game is a diagnostic tool not just to see if you can differentiate quickly but also if you can recall what has been revealed so far.

But this is as far as my positive feelings go for this game, because it is a speed game. A fun one initially as the pressure is on to react on the second showing and seek the token 'hidden' in the confusion of a messy pool. The latter feature of 'hide and seek' obviously diminishes as the pool 'dries' up. It is the speed feature I have the most issue with as the design won't allow you to handicap and so include slower players, plus there is the Jungle Speed issue of long nails.
